### 简介

```
一个视频下载汇聚类功能  
将视频下载地址推送给 应用目前已经支持的下载器 目前支持http 及外置Aria2 
并建立视频资源缓存 后台可进行管理查询查看等 
后期预计增加视频墙功能和APP端播放视频功能
```
